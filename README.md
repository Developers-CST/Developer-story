# Developer-story
 This repository narrates the journey of setting up effective developer collaboration, sharing real-life experiences and lessons while implementing communication, tools, and project management strategies.

## Shift left testing
_Shift-left testing is an approach to software testing and system testing in which testing is performed earlier in the lifecycle (i.e. moved left on the project timeline). It is the first half of the maxim "test early and often"._
